# Pinterest Style Website #

### Stack ###
*Ruby on Rails
*PostgreSQL
*Bootstrap
*AWS S3

### Functionality ###
Users can sign up for accounts, create pins, and upload pictures.  All pins display on main page and will dynamically resize based on the size of the window.
